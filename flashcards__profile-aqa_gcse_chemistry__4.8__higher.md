1. clean wire (e.g. Clean first to avoid contamination from previous ions.)
2. place sample on wire
3. place in blue / non-luminous flame (e.g. Use the blue Bunsen flame, not the yellow safety flame.)
4. observe flame colour

## How do you use an Rf value to calculate the distance moved by the solvent?

1. Rf = distance moved by substance / distance moved by solvent
2. distance moved by solvent = distance moved by substance ÷ Rf (e.g. Rearrange the formula so solvent distance = substance distance ÷ Rf.)

## How do you carry out paper chromatography to determine an Rf value?

1. draw a pencil start line on chromatography paper (e.g. Use pencil so the line does not dissolve.)
2. place a spot of sample on the start line
3. place solvent in a beaker
4. place the paper in the solvent with the start line above the solvent level (e.g. The spot must start above the solvent level.)
5. use a lid (e.g. Stops the solvent evaporating.)
6. wait for the solvent to travel up the paper
7. mark the solvent front (e.g. Mark the solvent front before it evaporates.)
8. dry the chromatography paper
9. measure distance from start line to centre of spot (e.g. Measure to the centre of each spot.)
10. measure distance from start line to solvent front
11. use the measurements to determine Rf value
